# AGENTS.md

## Projeto

Este projeto é a IA "Lia".

A Lia NÃO é apenas um chatbot.

Ela deve evoluir para uma assistente pessoal semelhante à Jarvis.

## Objetivos

- Nunca quebrar funcionalidades existentes.
- Sempre manter o projeto executável.
- Fazer alterações pequenas e incrementais.
- Preferir editar arquivos existentes em vez de criar novos.
- Escrever código limpo e comentado.
- Utilizar tipagem quando fizer sentido.
- Nunca colocar API Keys diretamente no código.

## Arquitetura

Fluxo principal:

Usuário
↓
Interface
↓
Assistente
↓
Interpretador
↓
Memória
↓
Conhecimento
↓
Pesquisa
↓
Resposta

## Pesquisa

No futuro a pesquisa utilizará:

- OpenAI
- Gemini
- DeepSeek
- Wikipedia
- Web

A resposta final deverá ser produzida pela própria Lia utilizando todas essas fontes.

## Regras

- Nunca apagar funcionalidades existentes.
- Nunca alterar nomes de arquivos sem necessidade.
- Sempre preservar compatibilidade.
- Sempre explicar alterações complexas em comentários.
